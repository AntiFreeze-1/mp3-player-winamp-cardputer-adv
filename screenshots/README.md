# 📸 Screenshots

Place your screenshots here!

## Recommended Screenshots:

1. **Main Interface** - Winamp player interface showing file list
2. **Playing Track** - Player with "PLAY" status and animated equalizer
3. **File Browser** - File list with highlighted current track
4. **Controls** - Close-up of control buttons

## File Naming:

Use descriptive names:
- `main_interface.jpg`
- `playing_track.jpg`
- `file_browser.jpg`
- `controls.jpg`

## Image Format:

- **Format:** JPG or PNG
- **Size:** Recommended 240x135 (native Cardputer resolution) or larger
- **Quality:** Good quality, but keep file size reasonable (< 500KB)

## Adding Screenshots to README:

After adding screenshots, update `README.md` to include them:

```markdown
## 📸 Screenshots

### Main Interface
![Main Interface](screenshots/main_interface.jpg)

### Playing Track
![Playing Track](screenshots/playing_track.jpg)
```
